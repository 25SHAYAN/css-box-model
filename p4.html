css box model

every element that can be displayed on a web page is comprised of one or more rectangular boxes. css box model typically describes how these rectangular boxes are laid out on web page this boxes have different properties and can interact with each other in diffrent ways,but every box has a content area and optional surounding padding, border and margin areas
